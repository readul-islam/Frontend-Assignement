## Installation step by step

## Clone the repository:

   ```bash
   git clone https://github.com/readul-islam/Frontend-Assignement
   ```
## For Frontend
   ```bash
   cd frontend
   ```

   ```bash
   npm install
   ```

   ```bash
   ADD .env File in root folder and add credentials
   ```

   ```bash
   npm run dev
   ```

## For Backend
   ```bash
   cd backend
   ```

   ```bash
   npm install
   ```

   ```bash
   ADD .env File in root folder and add credentials
   ```

   ```bash
   npm run dev
   ```
